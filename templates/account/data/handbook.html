2015.11月 颁布





欢迎致辞

       欢迎你来到致趣联媒（INAD），我们的前身是豆瓣的商务部，2014年我们这群共事多年且志趣相投的小伙伴一起从豆瓣独立出来创立了这家公司。我们曾经见证了中国互联网广告波澜壮阔的发展，这次我们又选择做移动互联网营销的弄潮儿，大家一起用兴趣和激情，要做一番有趣又有未来的事业。

       互联网从门户、搜索、视频、社交......到现在的移动互联网，这个行业的发展正在每天改变着我们的生活。当移动互联网遇到广告，会给这个传统行业带来怎样的变化？这是致趣人一直需要思考的。技术的不断进步也影响了广告行业的发展，社交、移动、程序化、大数据、实时交易等都是未来移动广告发展的技术趋势。而广告本身是一种艺术的表达，它从策略、创意、设计、执行都需要很强的创造力。致趣联媒希望可以整合互联网技术和广告艺术，把最神奇的移动互联网营销呈现给更多的媒体和品牌。

       公司秉承了豆瓣简单、平等、文艺的风格，又结合了专业、务实、高效的商业态度，混血的DNA造就了我们特殊的企业文化，Geek和吉他手一样受欢迎。清晰的公司战略和超强的团队执行力，让致趣在短时间内就成为行业里非常出众的公司，但我们离一家伟大的企业还相差甚远，在通向这个过程的道路上需要每一个小伙伴添砖加瓦。最后希望致趣能给你一个施展才华的平台，也希望你的到来，让我们的未来更加美好！

CEO:Oscar

公司简介

       在业界，致趣联媒是个新鲜的名字，但并不是个陌生的团队，我们孵化于豆瓣网广告部，基于豆瓣的兴趣社交平台，在兴趣营销领域有丰富的营销经验，团队由销售、策划、运营、设计、技术团队构成，在北京、上海、广州均有分公司，分别覆盖华北、华东、华南区域，总人数超过100人。

       致趣业务：打通优质客户与优质媒体（特别是移动端媒体）之间的桥梁，为品牌客户提供主要基于移动端的兴趣营销整合方案。

二、 致趣定位

致趣是中国首家优质新媒体联盟。

与其他代理、媒介公司不同，致趣在服务品牌、代理公司的同时，还服务于各类优质新媒体,。为媒体开发商业产品、制定销售策略、打通定位-开发-推广-策划-售卖-执行环节。致趣与国内优质新媒体一起，为品牌提供基于互联网,尤其是移动互联网的整合营销解决方案。

致趣的联合媒体矩阵日均可覆盖网民2.5亿。

三、 致趣特色

1、兴趣营销

致趣孵化于豆瓣网广告部，基于豆瓣的兴趣社交平台，在兴趣营销领域有五年的营销经验，通过阅读、音乐、电影、美食、展览、旅行等各类大众、小众的用户兴趣热点，链接品牌与用户。基于兴趣、设计兴趣化广告素材、定制兴趣化互动，在豆瓣书影音频道制作原生广告。联合音乐节、复古骑行等文娱线下活动，为品牌提供整合的兴趣解决方案。

致趣团队多次在艾菲奖、金投赏、长城奖、金鼠标奖等行业奖项中斩获殊荣。

2、服务优质品牌

致趣服务客户已超过三百家，涵盖IT、汽车、家电、快消、时尚、奢侈品等行业，如CHANEL、BURBERRY、大众、路虎、奥迪、宜家、MUJI、可口可乐、adidas、Converse、Intel、英国航空等。

3、整合优质媒介

致趣是豆瓣、知乎、陌陌、下厨房、课程格子、虎嗅等多家优质媒体的核心代理公司。通过整合优质的媒介资源，帮助品牌定位精准人群，并且通过致趣对优质媒介产品的开发定制组合，为品牌实现更深度的整合传播。

4、有趣有效有节操

保护用户体验，尊重媒体/产品的自有调性，使广告“有趣且有效”，是致趣团队一直以来的工作原则和目标。从素材设计到定制化创意互动，甚至是高度结合产品自身功能和内容的原生广告，以及通过兴趣标签、技术算法实现数据挖掘，。致趣坚持将“艺术”与“技术”结合，做让用户喜欢的广告，做有品质、有价值的营销。


工作关系

入职

1、 入职手续

人力资源的同事将会全程协助你办理入职手续

  • 所需个人资料：身份证原件、学历或学位证明原件、户口本首页及本人页复印件、原单位离职证明（加盖公章）、三个月内的有效体检报告、招商银行一卡通复印件（在复印件上手写姓名及卡号）；
  • 填写《基本信息登记表》；
  • 领取基本办公用品；
  • 开通工作邮箱及邮件组；
  • 人力资源的同事将带领你参观办公室，与部门负责人见面。
2、 把自己介绍给同事
  • 撰写个人介绍邮件发送至ALL邮件组；
  • 欢迎你在个人介绍邮件中展示你独特的个性、嗜好、星座、经历或宠物，让大家记住你；
  • 除个人介绍外，请附上你的生活照，让大家认识你。
3、 五险一金
  • 请告知人力资源你的五险一金是否在原单位转出；
  • 填写《团体保险被保险人个人事项告知书》。
4、 个人信息如有变化请及时告知
  • 当个人信息发生更改或补充时，请及时以邮件形式告知人力资源同事：姓名、住址、电话、户籍、婚姻状况、紧急联系人、培训结业或进修毕业证书、工资卡号等。

劳动合同

1、 你与公司的劳动合同将在入职一周内签订。
2、 劳动合同的变更
  • 你与公司协商确认后，签订《劳动合同变更书》
3、 劳动合同的续签
  • 续签流程：在劳动合同到期之日前30天，人力资源将发放书面《员工续签评估表》至你的直接主管处。直接主管将对你进行评估，结合你本人意愿，直接主管、部门负责人会在规定日期前将续签意见反馈给人力资源；
  • 决定续签：如果双方决定续签，人力资源将会与你签订一份新的劳动合同。其中续签劳动合同的期限，将由人力资源根据法律法规的要求，并参考部门的续签意见后直接决定；
  • 决定不续签：若公司决定不续签劳动合同，公司会在劳动合同到期之日前30天书面通知你办理相关离职手续；若你决定不与公司续签劳动合同，请在劳动合同到期之日前30天书面通知部门负责人及人力资源的同事，经双方确认后办理相关离职手续。
4、 劳动合同的解除和终止
  • 严重违反公司员工手册及规章制度、劳动纪律，累计收到公司书面警告三次及以上；
  • 严重失职，给公司造成重大损害，累计收到公司书面警告两次以上；
  • 营私舞弊或有重大品格或职业操守问题；
  • 利用职务便利获取、修改、泄露客户或非公开信息；
  • 以欺诈方式骗取公司信任订立或变更劳动合同；
  • 无故旷工3日以上；
  • 其它违反法律、行政法规的情况。

试用期及转正

1、 试用期
  • 试用期表现极优者可办理提前转正手续。试用期开始，你的直接主管或导师会向你介绍本部门职能、人员情况，讲解本职工作内容和要求，帮助你了解公司有关规章和规范，以及你将要开展工作的具体事务。
2、 转正流程
  • 试用期满三个月前1周，进入第一阶段的评估考核，你需要填写《试用期工作总结》；
  • 人力资源同事与部门负责人、工作导师及相关同事将会一起参与考核评定，确认予以提前转正及其他相关事宜；
  • 试用期结束前3-5个工作日，人力资源将向你发送试用期结果通知。

内部推荐流程

欢迎你推荐和你一样优秀的人才到公司，公司特设立“荐贤奖”。

推荐流程如下：

  • 可将候选人简历或个人信息直接发送至人力资源负责人邮箱；
  • 请文字说明推荐理由；
  • 推荐成功，顺利转正者，公司将给予一定的奖励：
    • • 推荐普通人员：奖金额度500-1000元
    • • 推荐部门经理及以上人员：奖金额度1000-2000元
    • • 推荐部门总监及以上人员：奖金额度2000-5000元
  • 以下人员不参加“荐贤奖”：公司管理层、人力资源部员工、被推荐岗位的上级；
  • 推荐有效期：一年。
    • • 具体招聘需求请随时浏览公司网站：http://www.inad.com/

离职

如果你最终决定离开致趣团队，我们希望继续保持友好和相互尊重的关系。希望你在离开前将离职手续办理完毕，将工作交接给相关同事。如有未清账目、公司物品，也请一并结清归还。

离职流程如下：

  1. 请先向你的部门负责人提交离职申请邮件，并说明缘由；已转正的员工离职请提前30个工作日提出申请，试用期员工请提前3个工作日提出申请；
  2. 请向人力资源领取《离职申请表》、《离职交接单》，进入具体的离职流程；
  3. 人力资源将向你开具《离职证明》。

薪资、福利、假期、和培训

薪资制度
  1. 基本制度：月薪制
  2. 年收入构成：13个月工资+奖金+各类补助
  3. 调薪：每年一季度月和三季度是薪酬回顾月
  4. 薪资发放：
    • • 当月10日发上月工资，如遇节假日则自动顺延；
    • • 你的工资卡为招商银行；
    • • 你的工资单于每月10日前以邮件形式收到；
  5. 薪酬保密原则：请不要透露你的薪资给同事，也不要打听同事的薪资。上述原则是致趣团队一直保持的的默契，请务必遵守。
保险与福利
  1. 基本保险与福利
    • • 基本社会保险：参考当地标准；
    • • 住房公积金：参考当地标准。
    • • 公司将为当月15日前（含15日）入职的同事于入职当月办理社保及住房公积金增员手续，15日后入职同事的社保及公积金增员手续将于下月15日前办理，相应社保及公积金费用于增员手续办理完成之月开始扣缴。公司将为当月15日前（含15日）离职的同事于离职当月办理社保及住房公积金减员手续，15日后离职同事的社保及住房公积金减员手续将于下月15日前办理，相应社保及住房公积金费用缴纳至减员手续完成之月；
  2. 其他福利
    • 电脑补助：如果你以自己的电脑（必须是笔记本电脑）作为办公电脑，那么你将享受公司给予的电脑补助200元/月。
    • 电话补助：职能部门助理级以下每人每月50元，助理级以上每人每月100元；业务部门每人每月200元。
    • 全勤奖：当月未出现任何迟到、早退、请假、旷工者，公司给予全勤奖金200元。
    • 宠物日：每周五，你可以带宠物来公司和你一起上班，公司将提供宠物食品和玩具；
    • 家庭日：每月第四个周五，欢迎你带家人来公司参观，公司将提供零食和舒适的环境；
    • 早餐：工作日的上午9：30-10：00；
    • 水果：工作日的下午15：00-15：30；
    • 结婚福利：如果你入职致趣后，领取了结婚证，除了婚假与祝福之外，还将有礼品赠送；
    • 生日祝福：生日当天除了收到祝福外，还有礼物赠送。
假期
  1. 公共假期
    • • 你可以享受国家法定的公共假期：元旦1天、春节3天、清明节1天、劳动节1天、端午节1天、中秋节1天、国庆节3天；
    • • 如果你已为人母，“六一儿童节“的上午可以休假半天陪伴小朋友。适逢节假日、双休日该休假不顺延；
    • • 公共假期无需申请，按公司统一规定执行，属于带薪假期，薪资全额发放。
  2. 年休假
    • • 你可按照该自然年内工作时长占全年的比例，享受所对应折算的年休假时间；
    • • 年休假以自然年为计算单位，即每年的1月1日-12月31日；每年年休假需在下一年的3月30日休完；
    • • 假期时长：不满1年，按入职时长比例乘以基数（10天）；1-5年，以10天为基础每年递增1天；5年以上，15天。计算方法：在职天数÷365×10=第一年年假天数，遇小数点小于0.5按照0天计算，大于0.5按照半天计算，以此类推；
    • • 如果你将离职，未休完年假可折算工资发放；
    • • 年休假属于带薪假期，你的薪资将会全额发放；
    • • 你可以通过内网查询自己的具体年休假情况。
  3. 病假
    • • 你入职之日起可享受每年6天的带薪病假；
    • • 请1天以内病假无需开具证明， 1天以上需提供医疗机构的诊断证明，试用期内月累计病假最长不超过5个工作日；
    • • 你的带薪病假休完后，病假工资按照当地最低工资的80％核发，需提供医疗机构诊断证明，病假期限最长不得超过3个月；
  4. 事假
    • • 你如果有私事办理需提前提出请假申请；
    • • 试用期内的员工月累计事假最长不超过3个工作日；
    • • 事假工资按照基本工资÷21.75×事假天数扣发。
  5. 婚丧假、产假

    婚假

    • • 在公司工作期间结婚，即可享受婚假。婚假的有效期为：正式注册登记结婚之日起1年内；
    • • 婚假时长：基础3天，晚婚10天。晚婚年龄为（夫妻双方同时满足）：男25周岁，女23周岁；
    • • 再婚员工可享受法定基础婚假（即3天），不能享受晚婚假。

    丧假

    • • 范围：直系范围；
    • • 时长：3天。

    产前假

    • • 女性员工怀孕，根据医院检查记录可享受十次的产前检查假，每次半天；
    • • 女性员工怀孕7个月以上的，可申请每周一天在家办公（直至休产假）；
    • • 根据各地区相关法律执行。

    产假

    在公司工作期间有生育情况，即可享受产假。

    • • 女性员工生育，可按国家规定享受产假，具体时长如下：

      基础98天，晚育增加30天(夫妻双方晚婚后生育或已婚妇女生育第一个子女时，年满24岁为晚育)；难产（含刨宫产）增加15天；多胞胎，多一胎增加15天；流产:妊娠不满4个月的，产假为20天；妊娠4个月以上的，产假42天（凭医学证明）。

    哺乳假

    • • 女性员工生育后且在其婴儿满1周岁（即哺乳期）前，每个工作日内可享受两次哺乳时间，每次30分钟。为方便执行，女性员工其婴儿哺乳期内的每个工作日可提前一小时下班。生育多胞胎的，多一胎每次哺乳时间增加30分钟，故下班时间以此类推。
    • • 婚丧假、产假均包含节假日和双休日，限一次性使用；
    • • 婚丧假、产假均属于带薪假期，工资全额发放；
    • • 婚丧假、产假不能折算成工资发放。
培训与分享
  1. 参加培训与分享之前
    • • 请先征得你的直接主管同意并确认；
    • • 请合理安排手头工作；
    • • 与培训内容无关的事物请安排在非培训时间进行；
    • • 请尊重讲师时间。
  2. 申请

考勤和请假

考勤制度

实行8小时工作制

  1. 弹性工作时间
    • • 上午： 9：00—12：00，下午：13：00——18：00；
    • • 上午： 9：30—12：30，下午：13：30——18：30；
    • • 上午： 10：00—13：00，下午：14：00——19：00。
  2. 门禁管理
    • • 出入办公区域需指纹打卡。
    • • 行政负责指纹录入，员工离职，需及时删除指纹记录。
  3. 出勤规定
    • • 上下班指纹打卡，月末汇总当月员工出勤情况，作为核算工资的标准；
    • • 因公外出而未能及时打卡者，须事先在内部系统填写外出报备。
  4. 迟到、早退
    • • 每月允许一次30分钟以内迟到，不扣款；
    • • 第二次迟到30分钟内，每次扣款30元，；30分钟以上按（日工资/小时）事假折算；
    • • 迟到、早退超过3次以上者，30分钟以内扣罚工资50元；
    • • 一个月内迟到、早退超过5次以上者，扣除当月绩效工资。
  5. 加班
    • • 公司不提倡加班，员工应充分利用8小时工作制，提高工作效率；
    • • 员工加班需向主管领导提出书面申请，加班超过2小时，补助加班餐费30元/人；加班4小时以上，可以选择打车回家，费用实报实销，允许第二天上午调休2小时，加班时间以打卡记录为准。（此条款不适用销售人员）
请假
  1. 年休假
    • • 休年假3天以内需提前一周向部门负责人申请；请假5天以上者，需提前一个月向部门负责人提出申请，CEO审批；
  2. 病假
    • • 请提供医院开具的病假单，病假期间工资计算将按照你工作所在地的法律法规执行。
  3. 如需请假请注意以下几点
    • • 请假前请你交接手头工作，并告知相应的协作同事；
    • • 如需续假或提前回公司上班，经部门负责人确认后，系统内续假或销假；
    • • 未经批准自行休假将视为无故旷工；
    • • 病假、事假、年休假等，均最少以半天计算。
旷工
  1. 无故旷工，将扣除相应的工资；
  2. 一年内无故旷工累计3次将收到书面警告；
  3. 一年内无故旷工超过5次，公司有权与其解除劳动合同。
